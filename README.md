# Andev

A cli tools for Android Developer

## Features

 * [x] Open deeplink 
 * [ ] Clear cache
 * [ ] Install apk
 * [ ] Uninstall package
 
## Install

```shell
$ npm i -g @talenguyen/andev
```

## Usage

```shell
Usage:
    $ andev --help

    Options:
      --deeplink, d Open deep-link
      --clear, c    Clear cache

    Example:
      $ andev --d https://google.com
      $ andev --c com.android.chrome
```
