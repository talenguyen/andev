# Andev

A cli tools for Android Developer

## Features

 * [x] Open deeplink 
 * [x] Clear cache
 * [x] Install apk
 * [x] Uninstall package
 * [ ] Send text to device
 * [ ] Connect device over WI-FI

## Install

```shell
$ npm i -g @talenguyen/andev
```
